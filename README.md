# AI Language Learning Platform

An AI-powered language learning platform that supports Japanese and over 119 languages worldwide.  
The system is designed to help users practice real-world language usage through interactive conversations and guided learning.

## Features
- AI-assisted language practice
- Supports Japanese and 119+ global languages
- Interactive conversation and Q&A
- Dynamic response generation based on user input
- Web-based access

## Tech Stack
- Frontend: VueJS / HTML / CSS
- Backend: API-based logic
- AI Integration: GPT-based language model
- Database: Firebase
